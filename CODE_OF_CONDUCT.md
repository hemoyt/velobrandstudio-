# Code of Conduct

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

Be respectful, assume good intent, and keep discussion focused on the work. Harassment, personal attacks, and spam are not tolerated. Maintainers may remove comments, commits, or contributors that violate this.

Report concerns by opening a private security advisory or contacting the maintainers directly.
